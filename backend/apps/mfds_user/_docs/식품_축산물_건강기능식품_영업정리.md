# 식품·축산물·건강기능식품 영업 등록·허가·신고 정리

> **구분 기준**
> - **허가** — 행정청 심사 후 허용 (재량 행위, 처리 30일 내외)
> - **등록** — 기준 갖추고 등록 (기속 행위, 처리 5~10일)
> - **신고** — 요건 갖추고 신고 (수리 불요 원칙, 즉시~3일)

---

## ① 식품위생법 영업

| 구분 | 업종 | 주요 내용 | 관할 / 근거 |
|------|------|-----------|-------------|
| **허가** | 식품조사처리업 | 방사선 시설·장비 기준 충족 필요 | 시·도지사 / 식품위생법 §37① |
| **등록** | 식품제조·가공업 | 시설기준, HACCP 적용 업종 별도 | 시·도지사 / 식품위생법 §37② |
| **등록** | 식품첨가물제조업 | 제조 품목별 기준·규격 준수 | 시·도지사 |
| **등록** | 기구 등 살균·소독업 | 살균소독제 제조·판매업 | 시·도지사 |
| **신고** | 즉석판매제조·가공업 (빵집, 떡집, 반찬가게 등) | 소분·판매에서 직접 조리 포함. 영업장 소재 신고 | 시·군·구청장 |
| **신고** | 식품운반업 | 냉장·냉동차량 기준 | 시·군·구청장 |
| **신고** | 식품소분업 | 완제품 소분·포장 판매 | 시·군·구청장 |
| **신고** | 식품판매업 (유통전문판매업, 식용얼음판매업, 기타 식품판매업) | 업종에 따라 시설기준 상이 | 시·군·구청장 |
| **신고** | 식품보존업 (식품냉동·냉장업, 식품조사처리업 소규모) | 냉동·냉장창고 시설기준 | 시·군·구청장 |
| **신고** | 집단급식소 설치·운영 (1회 50명 이상 상시 급식) | 학교·병원·사업장·기숙사 등. 영양사 의무 고용(일정 규모) | 시·군·구청장 |

> ※ 식품제조·가공업·식품첨가물제조업은 2020년 이전까지 '허가'였으나 현재는 '등록'으로 완화

## ② 축산물위생관리법 영업

| 구분 | 업종 | 주요 내용 | 관할 / 근거 |
|------|------|-----------|-------------|
| **허가** | 도축업 (소·돼지·닭 등 도살·해체) | 도축장 시설기준 엄격. 검사관 상주 필요 | 시·도지사 / 축산물위생관리법 §22① |
| **허가** | 집유업 (원유 수집·냉각·보관) | 집유시설·냉각탱크 기준 | 시·도지사 |
| **허가** | 축산물가공업 (햄·소시지·우유·치즈 등) | HACCP 의무 적용 대상 多. 제품 유형별 기준 | 시·도지사 |
| **허가** | 식육포장처리업 (식육 포장·냉장·냉동) | 포장처리실, 냉장·냉동시설 기준 | 시·도지사 |
| **신고** | 축산물운반업 | 냉장·냉동차량 기준, 위생관리 기준 준수 | 시·군·구청장 |
| **신고** | 축산물보관업 | 냉장·냉동창고 기준 | 시·군·구청장 |
| **신고** | 축산물판매업 (식육판매업·식육부산물전문판매업·우유류판매업·축산물유통전문판매업) | 판매업 종류별 시설기준 상이. 식육판매업은 냉장설비 필수 | 시·군·구청장 |
| **신고** | 식육즉석판매가공업 (정육점에서 직접 가공·판매) | 햄·소시지 등 소량 직접 가공 허용 | 시·군·구청장 |

> ※ 도축업·집유업·축산물가공업·식육포장처리업은 '허가' 대상으로 심사가 엄격하며 HACCP 의무화 비율 높음

## ③ 건강기능식품법 영업

| 구분 | 업종 | 주요 내용 | 관할 / 근거 |
|------|------|-----------|-------------|
| **허가** | 건강기능식품 제조업 | GMP 의무화(2020년~). 기능성 원료 인정 필요. 제조 품목 사전 신고 병행 | 시·도지사 / 건강기능식품법 §5① |
| **신고** | 건강기능식품 유통전문판매업 (수입·위탁제조 후 자체 브랜드 판매) | 제조는 외주, 판매만 자체 운영. GMP 제조업체와 위탁계약 필수 | 시·군·구청장 / 건강기능식품법 §5② |
| **신고** | 건강기능식품 판매업 (약국, 편의점, 인터넷 쇼핑몰 등) | 별도 시설기준 없음. 영업자 교육 이수 후 신고 | 시·군·구청장 |

> ※ 건강기능식품 제조업은 GMP(우수건강기능식품 제조기준) 적합 인정 필수 — 미인정 시 허가 불가

---

## ④ 허가·등록·신고 핵심 비교

| 구분 | 허가 | 등록 | 신고 |
|------|------|------|------|
| **행정 성격** | 재량 행위 (심사 후 허용) | 기속 행위 (기준 충족 시 등록) | 수리 불요 원칙 (접수로 효력) |
| **처리 기간** | 30일 내외 (현장 검사 포함) | 5~10일 | 즉시~3일 |
| **주요 요건** | 시설·인력·설비 전면 심사 | 등록 기준 충족 서류 제출 | 시설·교육 요건 확인 |
| **취소·정지** | 허가 취소 가능 | 등록 취소 가능 | 영업 정지·폐쇄 명령 |
